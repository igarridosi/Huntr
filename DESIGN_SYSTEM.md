# HUNTR - DESIGN SYSTEM & UI GUIDELINES

Brand Personality: "The Wolf of Value Street".
Vibe: Precisión, Instinto, Naturaleza Salvaje, Premium Dark Mode.

## 1. PALETA DE COLORES (THE WOLF PALETTE)
Usa estos colores exactos en la configuración de Tailwind:

- **Background (Deep Forest Night):** `#0B1416` (Negro verdoso muy oscuro) - Para el fondo principal.
- **Surface (Midnight Rock):** `#162225` (Un tono más claro) - Para tarjetas y paneles laterales.
- **Primary Accent (Sunset Orange):** `#FF8C42` (Naranja vibrante) - Para botones de acción (CTA) y gráficos alcistas.
- **Secondary Accent (Golden Hour):** `#FFBF69` (Amarillo anaranjado suave) - Para estados de alerta o destacados.
- **Text Primary (Snow Peak):** `#F2F4F3` (Blanco roto/crema muy claro) - Para legibilidad máxima.
- **Text Secondary (Mist):** `#8C9DA1` (Gris azulado) - Para etiquetas y metadatos.

*Regla:* NO uses verde/rojo estándar para la bolsa.
- Usa el **Naranja (#FF8C42)** para subidas (Bullish/Heat).
- Usa un **Gris neutro o Azul apagado** para bajadas. Mantén la estética cálida.

## 2. TIPOGRAFÍA
- **Headings:** `Satoshi` o `Outfit` (Sans-serif con carácter).
- **Data/Numbers:** `Geist Mono` o `JetBrains Mono` (Tabular nums para alineación financiera perfecta).

## 3. COMPONENTES CLAVE (UI PATTERNS)
- **Cards:** Bordes redondeados (`rounded-xl`). Sin bordes visibles, usa una sombra sutil o un borde muy fino en color `#2A3B40`.
- **Gráficos (Tremor/Recharts):**
  - Líneas finas.
  - Gradientes de relleno usando el Naranja hacia transparente.
  - Tooltips oscuros con el fondo `#0B1416`.
- **Search Bar (The Hunter's Eye):**
  - Grande, central o accesible con `Cmd+K`.
  - Estilo "Glassmorphism" muy sutil sobre el fondo oscuro.

## 4. REGLAS DE "VIBE CODING"
- Evita el diseño "Bootstrap". Esto debe sentirse como una herramienta táctica moderna.
- Espaciado generoso (Whitespace). Que los datos respiren.
